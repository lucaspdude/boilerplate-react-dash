## FrontEnd GoBarber desenvolvida com as tecnologias:

- ReactJs
- Redux
- Redux-Saga
- Styled-Components
- Redux-Persist
- Yup
- Axios
- Immer
- Entre outras



## Para iniciar o projeto

Adicionando Repositorios:

```js
yarn
```


Iniciando a aplicação:

```js
yarn start
```
